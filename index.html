Mekanisme Web API (OMDb)

Aplikasi ini mendemonstrasikan cara kerja Fetch API dengan langkah-langkah berikut:

  • 1. Request: Browser mengirim permintaan HTTP GET ke endpoint https://www.omdbapi.com/.
  • 2. Parameter: URL menyertakan apikey=[API_KEY] (untuk otentikasi) dan s=[judul] (query pencarian).
  • 3. Asynchronous: Kode menggunakan async/await untuk menunggu respon server tanpa membekukan antarmuka pengguna.
  • 4. Response: Server OMDb mengembalikan data dalam format JSON.
  • 5. DOM Manipulation: JavaScript memproses JSON tersebut dan membuat elemen HTML (kartu film) secara dinamis.
